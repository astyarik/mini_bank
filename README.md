# mini bank
Небольшой искусственый банк, где есть возможность переводить, работать, смотреть баланс, выйти или зайти в админ мод
В админ мод вы можете снять или добавить деньги игроку по ИМЕНИ (не ivan, jhon итд, а по Иван, Джон (итд)). Либо добавить нового пользователя
